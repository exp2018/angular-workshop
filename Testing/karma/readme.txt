1. Running the App [in a command console]
		> npm start
	
2. Run the Unit Tests [in another command console] 
		> npm test
		
3. Check Test Coverage [in another command console]
   (to work coverage the step 2/Karma must be ran in 'singleRun' mode; see karma.conf.js): 
		> npm run test:once
		> npm run coverage
